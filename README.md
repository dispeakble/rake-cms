# Rake CMS

A modern content management system like WordPress, built with Next.js.

**Stack:** Next.js 15 (App Router) · TypeScript · Tailwind CSS · PostgreSQL (via Prisma ORM)

## Overview

Rake CMS aims to provide the ease of use of WordPress with the modern performance, security, and developer experience of the Next.js ecosystem. Designed for content creators and developers who want a headless-capable, fully customizable CMS backend.
